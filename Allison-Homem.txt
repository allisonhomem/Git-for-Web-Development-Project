    1. What is Semantic HTML? 
 HTML tags that are descriptive.
    2. What is HTML used for? 
 To mark-up the content for a webpage.
    3. What is an attribute and where do we put it? 
 It provides information about the tag, and goes inside the opening tag.
    4. What is the h1 tag used for? How many times should I use it on a page?
 The most important word/phrase on the page. It should only occur once.
    5. Name two tags that have required attributes
 <img> and <a> : image and anchor tags
    6. What do we put in the head of our HTML document? 
 Document type, title and other meta information and style sheets.
    7. What is an id? 
 An attribute that gives a unique name to an element, referred to later for style purposes in CSS.
    8. What elements can I add an id to? 
 Any elements.
    9. How many times can I use the same id on a page? 
 Once.
    10. What is a class? 
 Another way of naming an element, but may be used for more than one element.
    11. What elements can I add a class to? 
 Any elements.
    12. How many times can I use the same class on a page? 
 As many as you like.
    13. How do I get my link to open in a new tab?
 By adding target="_blank" to the anchor tag.
    14. What is the alt attribute in the image tag used for? 
 Gives a description of the image that will appear if the image fails to load.
    15. How do I reference an id?
 In CSS you can reference an id by #idname, and in JavaScript you can .getElementById('idname').
    16. What is the difference between a section and a div?
 Both block off a group of content and can be used to style that content, but section is more descriptive and can be used by search engines to find your page.
    17. What is CSS used for? 
 To style the webpage and make it pretty.
    18. How to we select an element? Example - every h2 on the page
 h2{} ...by its element name.
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
 A class can be used for multiple elements, and in id can only be used once. Use a class when you want to style many elements with the same style, and id when you want to style one element specifically, or call it in JavaScript.
    20. How do we select classes in CSS?
 .classname or element.classname 
    21. How do we select a p element with a single class of “human””?
 p.human
    22. What is a parent child selector? When would this be useful? 
 parentelement childelement ...it applies style to all named child elements inside the named parent element. It would be useful when you want the same style for all parent child pairs of those element types named.
    23. How do you select all links within a div with the class of sidebar?
 div.sidebar a 
    24. What is a pseudo selector?
 A keyword added to a selector that specifies a certain state in which the style will be applied.
    25. What do we use the change the spacing between lines?
 line-height
    26. What do we use to change the spacing between letters?
 letter-spacing 
    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
 text-transform 
    28. How do I add a 1px border around my div that is dotted and black?
 div { border: 1px dotted black;}
    29. How do I select everything on the page? 
 *{}
    30. How do I write a comment in CSS?
 /* */
    31. How do I find out what file I am in, when I am using the command line? 
 pwd
    32. Using the command line - how do I see a list of files/folders in my current folder?
 ls 
    33. How do I remove a file via the command line? Why do I have to be careful with this? 
 rm ...it completely deletes the file from your computer, there is no recycle bin or undo.

 When you have finished the file you are working on.

 git push -u origin branch-name

 Fork the repo, then clone, then git clone <repo link> in command line, then branch, work on project, then git add . your work, then git commit -m with comments, then git push -u filename, then submit pull request in github.
